# Cypress Helpers

## Installation

Install package

```
npm install @baongoc/cypress-helpers
```

## Usage

Import custom commands in `cypress/support/index.js` file:

```
import "@baongoc/cypress-helpers";
```

Import custom functions:

```
import {
  stringHelpers,
  numberHelpers,
  formHelpers,
} from "@baongoc/cypress-helpers";
```
